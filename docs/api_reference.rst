API Reference
==============

This page provides detailed documentation for all public functions and classes organized by subpackage.

The API is organized into three main subpackages:

* :doc:`polygeohasher <api_reference/polygeohasher>` - Geohash-based polygon subdivision and visualization
* :doc:`utils <api_reference/utils>` - Low-level utility functions for geohash operations, GADM downloads, and polygon processing
* :doc:`data <api_reference/data>` - Geospatial data collection from multiple sources (Copernicus, ERA5, CHELSA)
